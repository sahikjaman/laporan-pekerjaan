# Laporan Pekerjaan Lapangan

Sistem manajemen laporan pekerjaan lapangan dengan mode kolaborasi tim.

## Fitur

- Dashboard statistik real-time
- Tambah, edit, hapus laporan
- Filter dan pencarian
- Mode kolaborasi tim
- Data tersinkronisasi untuk semua pengguna

## Cara Install & Jalankan Lokal

1. Clone repository
2. Install dependencies: `npm install`
3. Jalankan: `npm run dev`
4. Buka browser: `http://localhost:5173`

## Deploy

Deploy otomatis via Vercel/Netlify/GitHub Pages
